# Provisionar um cluster de kubernetes utilizando k3d em terraform

## Requisitos mínimos

- Terraform >= 1.0.0
- Docker 
- Kubectl
- K3d (Opcional) - *Será instalado durante a execução do terraform*

## Utilizando..

git clone
cd k8s_/scripts
chmod +x install_k3d.sh

cd .. && terraform plan
terraform apply
